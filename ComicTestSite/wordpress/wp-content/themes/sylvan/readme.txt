== Description ==
Sylvan is a visually striking theme that evokes the mood of the forest, and of log cabins, and all things rustic.  Perfect for a nature-themed blog, it is responsive to look good on all screens.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Sylvan includes support for Infinite Scroll in Jetpack.
